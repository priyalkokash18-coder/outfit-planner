# 👗 Outfit Planner
Outfit Planner is a simple web application that helps users discover outfit ideas based on their preferences.
The user enters details such as:
- Age
- Occasion
- Budget
Based on the inputs, the application suggests suitable outfit options and styling ideas.
---
## ✨ Features
✔️ User-friendly interface  
✔️ Outfit suggestions based on occasion  
✔️ Budget-based recommendations  
✔️ Responsive and simple design  
✔️ Interactive experience using JavaScript  
---
